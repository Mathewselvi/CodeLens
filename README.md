# AI Code Insight

A complete production-ready MERN stack application designed to analyze GitHub repositories, website URLs, and ZIP source code uploads. 

The application detects:
1. Programming languages used
2. Percentage of code likely generated by AI (probability scoring)
3. Bugs, errors, and code quality issues
4. Security vulnerabilities
5. Performance bottlenecks
6. Improvement suggestions

## Tech Stack
- Frontend: React.js, TailwindCSS (v4), Axios, React Router, Chart.js
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)

## Core Structure
- \`client/\` - Vite/React application with modern dynamic dashboard components.
- \`server/\` - Express backend implementing robust pattern detection logic.

## Setup Instructions

1. **Prerequisites**
   Have Node.js (v18+) and MongoDB installed and running on your local machine.

2. **Backend Setup**
   \`\`\`bash
   cd server
   npm install
   npm run start
   \`\`\`
   *(Ensure MongoDB is running locally on port 27017)*

3. **Frontend Setup**
   \`\`\`bash
   cd client
   npm install
   npm run dev
   \`\`\`

4. **Access the App**
   Open your browser and navigate to \`http://localhost:5173\`.

## Features Delivered
- **Analyzers**: GitHub repos (cloning), Websites (downloader), ZIP uploads
- **Code Engine**: Evaluates Python, JS, HTML, CSS, Java, Go, PHP, C#, Ruby, TypeScript
- **Output Report Formatting**: Includes the strictly formatted Raw Text Report + Interactive dynamic web charts
- **Aesthetic**: Uses Premium glassmorphism effects, gradient typography, and Outfit typeface.
